# BSMS - BookStore Management System

MERN Project: A role-based web application enabling Readers, Shop Owners, and Admins to interact within a unified platform. Users can explore books, manage inventory, and handle transactions with features like authentication, catalog browsing, and scalable modular architecture built using MongoDB, Express, React, and Node.js.

## Tech Stack
- **Frontend**: React + Vite
- **Backend**: Node.js + Express
- **Database**: MongoDB
- **UI**: Bootstrap + Lucide React

## Getting Started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
